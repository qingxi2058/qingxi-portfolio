🖥️ 如何设为电脑壁纸
把这个动态倒计时变成你的桌面壁纸，每次开机都能看到，完全免费。
🍎 macOS 用户
  1. 在 App Store 免费下载安装 Plash
  2. 打开 Plash → 点击菜单栏图标 → Preferences
  3. 把下方的网址粘贴到 URL 输入框
  4. 勾选「在屏幕下方显示」，壁纸立即生效 ✅
🪟 Windows 用户
  1. 在 Microsoft Store 免费下载 Lively Wallpaper
  2. 打开 Lively → 点击「+」添加 → 选「网页」
  3. 把下方的网址粘贴进去，点击「确定」
  4. 选中后点「设为壁纸」✅
💡 壁纸页会自动读取你在主页设置的下班时间，无需重新配置。建议先在主页设好下班时间再设壁纸。
